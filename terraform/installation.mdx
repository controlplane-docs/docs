## Installation Instructions(#installation) 

The **Verified** Control Plane Provider is available from the [Terraform Registry](https://registry.terraform.io/).

Click [here](https://registry.terraform.io/providers/controlplane-com/cpln/latest) to obtain the latest version.

To install the provider, follow the directions in the link above, and then run `terraform init`.

**NOTE:** Requires Terraform 0.13+

## [Terraform Links](#links)

- [Download Terraform CLI](https://www.terraform.io/downloads.html)

## [Upgrading Provider Version](#upgrading-provider-version)

After installing a new provider version, follow these steps to upgrade an existing installation:

1. Update the version property inside the HCL file that contains the `required_providers` declaration block
   for the `cpln` provider.

2. Run the command: `terraform init -upgrade` to upgrade the Terraform dependencies (state file, etc.). The
   Terraform commands (plan, apply, etc.) will now use the latest version.

<Note>
If necessary, the provider version can be downgraded.
</Note>
