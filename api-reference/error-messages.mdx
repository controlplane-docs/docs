## [Workload Deployment Errors](#workload-deployment-errors)

- `Error: exitCode: 1 message: standard_init_linux.go:228: exec user process caused: exec format error`
  - The incorrect platform was used to build the configured image. Control Plane requires the image target `amd64`.
    Refer to the [Push an Image Guide](guides/push-image#step-2) for additional details.
